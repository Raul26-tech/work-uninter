# Portfólio Pessoal - Raul Santos

Projeto desenvolvido para a atividade prática de Fundamentos da Programação Web.

## Tecnologias utilizadas

- HTML5
- CSS3 puro
- JavaScript puro

## Estrutura

- `index.html`: página única com as seções obrigatórias.
- `css/style.css`: estilos do projeto.
- `js/script.js`: interações, tema e validação do formulário.

## Seções

- Sobre mim
- Formação
- Portfólio
- Contato

## Observação

O projeto não utiliza Bootstrap, Tailwind, React, Vue, Angular, jQuery ou qualquer framework/biblioteca externa.
